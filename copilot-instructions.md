# Password Input Enhancements

When working on password input fields, follow these guidelines:

- **Show/Hide Toggle**: Always include a toggle button to show or hide the password text.
- **Animations**: Add smooth transitions or animations for the toggle action to improve user experience.
- **Styling**: Use modern, accessible styling for the input field and toggle button.
- **Password Strength Indicator**: Include a visual indicator showing password strength (weak, medium, strong) with color coding.
- **Accessibility**: Ensure the toggle is keyboard accessible and screen reader friendly.
- **Icons**: Use appropriate icons (eye open/closed) for the toggle button.

Apply these enhancements to all password fields in the project.
