# 🔐 Secure Password Input Web App

A simple **Secure Password Input Web Application** built using **HTML, CSS, and JavaScript**.  
This project demonstrates password validation, secure input handling, and interactive frontend logic for learning and practice.

---

## 🚀 Features

- Secure password input field
- Password visibility toggle
- Basic password validation
- Clean and responsive UI
- Built using vanilla JavaScript (no frameworks)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
